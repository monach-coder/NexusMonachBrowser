# Nexus Monach Browser 2.7

Nexus Monach — лёгкий браузер для Windows 10/11 на базе актуального Chromium-движка Microsoft Edge WebView2. Код создан с нуля. У браузера собственный интерфейс и круглая эмблема, нет нашей телеметрии, а сбалансированная защита от отслеживания включена по умолчанию.

## Что уже работает

- вкладки с ленивой загрузкой, экономией памяти и первым этапом Semantic Tab Decay: неактивные вкладки плавно выцветают, а затем приостанавливаются;
- нормальная адресно-поисковая строка и отдельная кнопка поиска;
- DuckDuckGo, Brave Search, Startpage, Google и Яндекс на выбор;
- кнопки назад, вперёд, обновить/остановить, домой, свернуть, развернуть и закрыть;
- закладки, загрузки и восстановление вкладок; отдельный список истории удалён в пользу графа знаний;
- отдельные приватные окна через настоящий InPrivate-профиль WebView2;
- уровни защиты Basic / Balanced / Strict;
- выделенная сетевая строка высотой 40 px: TLS, маршрут, текущий сайт, число запросов, порты, сторонние узлы и блокировки; дублирующее окно Shield удалено;
- сетевая строка показывает текущую скорость загрузки/отправки активного адаптера и локальный `PING LAN` до шлюза Windows без обращения к внешнему серверу; при разворачивании окно ограничивается рабочей областью Windows и не прячет строку за панелью задач;
- скорость и ping окрашиваются от зелёного к бордовому; отсутствие трафика показано нейтральным цветом, потому что простой страницы не является ошибкой сети;
- защита загрузок: предупреждение для исполняемых файлов и двойных расширений, репутационная проверка WebView2, SHA-256 и сведения о сертификате подписи;
- локальная защита от фишинга: смешанные алфавиты, подмена известных доменов, Punycode, IP-адреса, HTTP и нестандартные порты;
- контекстный автономный AI-компаньон работает как исследователь сайта и сравнитель товаров; Ollama и сетевой API удалены;
- перевод видимого текста непосредственно в структуре страницы: заголовки, кнопки, пункты меню и абзацы остаются на своих местах; компактная панель позволяет вернуть оригинал;
- перевод существующей дорожки субтитров активного HTML5-видео на русский с сохранением таймкодов и показом внутри видео, включая полноэкранный режим;
- штатный перевод звука видео: сначала Chromium захватывает аудиопоток активного HTML5-видео; если плеер не отдаёт дорожку, после явного запуска используется краткий локальный loopback-захват системного выхода Windows. Встроенный whisper.cpp распознаёт речь, Nexus Fast Intelligence переводит её, а Nexus показывает русские реплики снизу видео;
- whisper.cpp и многоязычная модель `base-q5_1` поставляются в Full Offline release и никогда не загружаются при работе браузера;
- Smart Capsules: локальная модель группирует открытые вкладки по смыслу, а архивация сохраняет их в граф знаний и освобождает панель вкладок;
- Knowledge Constellation: просмотренные несекретные страницы становятся узлами локальной памяти, связываются по смыслу, общему источнику и реальным переходам; поиск использует multilingual-e5-small, карта показывает темы, силу связей, маршруты, посещения, закреплённые узлы и позволяет повторно открыть страницу;
- локальный диспетчер задач распределяет речь, семантику, зрение и текстовый анализ между специализированными моделями, загружая их только при необходимости;
- Passkeys, Windows Hello и FIDO2 через системную реализацию WebAuthn без доступа браузера к биометрическим данным;
- встроенное расширение `Nexus Monach DevTools AI` добавляет вкладку `Nexus AI` прямо внутрь Chromium DevTools: анализирует выбранный `$0`, безопасную HAR-сводку, отвечает на вопросы и по команде подсвечивает найденный элемент страницы;
- DevTools AI ничего не нажимает и не изменяет самостоятельно;
- DevTools AI не получает cookies, пароли, токены, значения полей ввода, тела запросов и право на произвольное исполнение JavaScript;
- на верхней панели кнопка `RU` переводит страницу, существующие субтитры или непосредственно звук видео;
- кнопка `AI` открывает исследователь сайта: он принимает текст либо фотографию/рисунок, собирает результаты, прокручивает страницу и переходит максимум по пяти страницам того же сайта;
- агент не покидает исходный домен, не добавляет товары в корзину и не оформляет заказ; неизвестные значения помечаются как «нет данных», а ссылки принимаются только из реально извлечённых результатов;
- собственные вспомогательные окна, AI-компаньон, основное меню и подсказки получили единый бесцветный полупрозрачный фон с 70% непрозрачностью, закруглённые углы и полностью непрозрачный чёткий текст;
- при запуске на splash-экране звучит короткий локально синтезированный аккорд, после чего Nexus пытается выбрать доступный в Windows мужской SAPI-голос для произношения «Nexus Monach»;
- верхние системные кнопки уменьшены; кнопка закрытия заменена безопасным перезапуском с сохранением вкладок, а закрытие оставлено системной команде панели задач Windows;
- компактная нижняя панель Privacy Dock работает без внешних диагностических запросов: локальные интерфейсы, маршрут, VPN/Proxy/Tor-порты, WebRTC без STUN/TURN, DNS Windows, часовой пояс, язык и локальный идентификатор отпечатка;
- полноширинная схема сетевой топологии «Nexus Monach → системный маршрут → прокси → интернет → текущий сайт»;
- каждый узел топологии открывает расширенные сведения: сетевые адаптеры, DNS, шлюзы, прокси, локальные слушающие порты и фактические соединения сайта;
- локальные TCP/UDP-порты и владеющие ими процессы читаются напрямую из Windows IP Helper API (`iphlpapi.dll`) с резервным .NET-механизмом;
- текущий сайт показывает сторонние получатели запросов, заблокированные трекеры и реально использованные порты без активного сканирования чужих серверов;
- нижняя строка сетевого статуса с TLS, маршрутом, числом запросов, сторонних узлов и блокировок;
- новая вкладка создаётся левым кликом по пустому месту справа от вкладок и вставляется после активной; отдельная кнопка `+` удалена;
- на стартовой странице удалена дублирующая поисковая форма — осталась одна верхняя адресно-поисковая строка;
- собственный HTTP или SOCKS5-прокси браузера и список доменов, подключаемых напрямую без него;
- Do Not Track и Global Privacy Control;
- удаление `utm_*`, `fbclid`, `gclid`, `yclid` и других рекламных параметров из ссылок;
- дополнительная блокировка известных сторонних аналитических узлов в строгом режиме;
- запрос разрешения перед доступом сайта к камере, микрофону, геолокации и буферу обмена;
- блокировка уведомлений по умолчанию;
- импорт, включение, выключение и удаление распакованных Chromium-расширений;
- переносимый режим: настройки и профиль хранятся в папке `Data` рядом с программой;
- локальная стартовая страница, которая не обращается к внешним серверам.

## Почему выбран WebView2

Движок WebView2 использует актуальную платформу Chromium и обновляется вместе с Evergreen Runtime. Windows 11 уже содержит этот Runtime. Сам движок не дублируется внутри каждого браузерного архива, поэтому оболочка получается заметно легче Electron-сборки. Стандартный User-Agent Edge намеренно не подменяется: случайная подмена отпечатка повышает уникальность пользователя и иногда вызывает проверки сайтов.

## Честные ограничения

Полная анонимность и полная совместимость одновременно недостижимы. Nexus Monach уменьшает отслеживание, но сам по себе не скрывает внешний IP-адрес, операционную систему, TLS-отпечаток и поведение пользователя. Для сокрытия IP нужен отдельно выбранный VPN либо Tor.

Google официально запрещает часть OAuth-авторизации во встроенных WebView, включая WebView2. Поэтому отдельные входы «Войти через Google» могут потребовать системный браузер. Обходить эту защиту подменой идентичности браузера проект не будет. Если абсолютная совместимость с такими входами обязательна, следующий технический уровень — собственная сборка полного Chromium; она значительно тяжелее и сложнее в обновлении.

WebView2 поддерживает распакованные расширения, но не является полным Chrome. Расширения, завязанные на кнопку `chrome.action`, закрытые API Chrome Web Store или специфический интерфейс Chrome, могут работать частично. Контент-скрипты, фоновые обработчики и сетевые правила — наиболее подходящие типы дополнений.

Защита от фишинга сочетает локальные эвристики Nexus Monach и репутационную проверку движка. Она снижает риск, но не заменяет внимательную проверку адреса, особенно перед вводом пароля или платёжных данных. SHA-256 позволяет сверить загруженный файл с контрольной суммой издателя; наличие сертификата само по себе не означает, что файл безопасен.

## Автономный Nexus Intelligence Fabric

В браузере нет зависимости от Ollama и нет облачного резервного подключения. Full Offline release содержит собственные runtime и модели в папке `AI`: Qwen3 0.6B для быстрых структурированных текстовых задач, Whisper base q5_1 для речи, multilingual-e5-small для смыслового индекса и SmolVLM 500M для изображений. Runtime-код не содержит адресов загрузки моделей.

Кнопка `RU` открывает три режима: перевод текста непосредственно на странице, перевод существующей `subtitles/captions` дорожки и перевод звука. Звуковой режим работает фрагментами: встроенный whisper.cpp распознаёт речь, после чего встроенная текстовая модель переводит текст. Защищённое DRM-видео или проигрыватель, не предоставляющий HTML5-аудиопоток, может запретить захват.

Данные извлекаются только после явного действия пользователя и ограничиваются по объёму. Страница считается недоверенным источником: найденные внутри неё инструкции не управляют моделью. DevTools AI не получает значения полей, cookies, токены и тела запросов. Между расширением DevTools и моделью работает ограниченный loopback-мост самого Nexus; внешних AI-сервисов нет.

Стандартное окно Chromium DevTools создаёт Microsoft Edge Runtime, поэтому Nexus не меняет общий фон нативного окна. Вместо внешней декоративной панели встроенное расширение создаёт внутри DevTools собственную вкладку `Nexus AI` через Chrome DevTools Panels API. Если конкретная версия WebView2 заблокирует этот API, обычные DevTools продолжат работать и браузер покажет причину загрузки расширения.

## Монитор приватности и прокси

Privacy Dock занимает всю нижнюю ширину обычного окна браузера и обновляется раз в пять минут. Панель можно временно скрыть кнопкой `×` либо включать и выключать через меню и настройки. Проверка работает локально: читает интерфейсы, шлюз и DNS Windows, ищет локальные признаки VPN/Tor/Proxy и собирает ICE-кандидаты WebRTC без STUN/TURN. Внешний IP, ASN и его достоверный регион намеренно не запрашиваются; региональная подпись — лишь оценка по настройкам Windows.

Full Offline release уже содержит whisper.cpp и `ggml-base-q5_1.bin`. Браузер не скачивает их при первом запуске и не создаёт сетевой зависимости для перевода звука.

Прокси задаётся как SOCKS5 либо HTTP с адресом и портом. Изменение требует перезапуска Nexus Monach. Список прямых исключений действует только на этот встроенный прокси. Он не способен отменить маршрут полноценного системного VPN: такое исключение необходимо настраивать в самом VPN-клиенте. Ограничение WebRTC от непроксируемого UDP включено по умолчанию; его можно отключить ради совместимости с отдельными видеозвонками и P2P-сервисами. Nexus Monach намеренно не подменяет часовой пояс, язык или User-Agent под географию IP.

## Сборка переносимой версии

Требования:

1. Windows 10/11 x64.
2. [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
3. Microsoft Edge WebView2 Runtime (в Windows 11 уже установлен).

Маленький source-архив не содержит сотни мегабайт весов моделей. Для пользовательской публикации собирается отдельный Full Offline архив с папкой `AI`; обычная исходная сборка явно предупредит, если этот комплект отсутствует.

Запустите `RUN_BUILD.cmd`. Скрипт создаст:

```text
dist/NexusMonach-Portable/
dist/NexusMonach-Portable-win-x64.zip
```

Архив самодостаточный по .NET: устанавливать .NET на компьютер пользователя не потребуется. WebView2 Runtime остаётся системным и автоматически получает исправления безопасности.

Для разработки:

```powershell
dotnet restore NexusMonach.sln
dotnet run --project src/NexusMonach/NexusMonach.csproj
```

## Хранение данных

- portable-сборка: `NexusMonach-Portable/Data`;
- обычный запуск из Visual Studio: `%LOCALAPPDATA%\NexusMonach`;
- граф знаний: `knowledge-graph.json` в той же локальной папке `Data`; в приватном окне индексация не выполняется;
- приватное окно использует режим `IsInPrivateModeEnabled` и не записывает граф знаний и сессию;
- включение «Очищать данные сайтов при выходе» удаляет cookies, кэш и сохранённую сессию при закрытии обычного окна.

  Assets/        эмблема и локальная стартовая страница
