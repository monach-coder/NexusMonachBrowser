# Nexus Monach Browser 2.8.0

Nexus Monach — лёгкий браузер для Windows 10/11 на базе актуального Chromium-движка Microsoft Edge WebView2. Код создан с нуля. У браузера собственный интерфейс и круглая эмблема, нет скрытой телеметрии, а сбалансированная защита от отслеживания включена по умолчанию. Очищенный crash-report отправляется только в явно выбранном пользователем режиме; без HTTPS-приёмника он остаётся локальным.

  анализа, выжимку и проверяемые адреса релевантных разделов этого сайта;

- верхние системные кнопки уменьшены; кнопка закрытия заменена безопасным перезапуском с сохранением вкладок, а закрытие оставлено системной команде панели задач Windows;

- запрос разрешения перед доступом сайта к камере, микрофону, геолокации и буферу обмена;

- импорт, включение, выключение и удаление распакованных Chromium-расширений;
- переносимый режим: настройки и профиль хранятся в папке `Data` рядом с программой;
- локальная стартовая страница, которая не обращается к внешним серверам.

## Почему выбран WebView2

Движок WebView2 использует актуальную платформу Chromium и обновляется вместе с Evergreen Runtime. Windows 11 уже содержит этот Runtime. Сам движок не дублируется внутри каждого браузерного архива, поэтому оболочка получается заметно легче Electron-сборки. Стандартный User-Agent Edge намеренно не подменяется: случайная подмена отпечатка повышает уникальность пользователя и иногда вызывает проверки сайтов.

## Честные ограничения

Полная анонимность и полная совместимость одновременно недостижимы. Nexus Monach уменьшает отслеживание, но сам по себе не скрывает внешний IP-адрес, операционную систему, TLS-отпечаток и поведение пользователя. Для сокрытия IP нужен отдельно выбранный VPN либо Tor.

Google официально запрещает часть OAuth-авторизации во встроенных WebView, включая WebView2. Поэтому отдельные входы «Войти через Google» могут потребовать системный браузер. Обходить эту защиту подменой идентичности браузера проект не будет. Если абсолютная совместимость с такими входами обязательна, следующий технический уровень — собственная сборка полного Chromium; она значительно тяжелее и сложнее в обновлении.

WebView2 поддерживает распакованные расширения, но не является полным Chrome. Расширения, завязанные на кнопку `chrome.action`, закрытые API Chrome Web Store или специфический интерфейс Chrome, могут работать частично. Контент-скрипты, фоновые обработчики и сетевые правила — наиболее подходящие типы дополнений.

Защита от фишинга сочетает локальные эвристики Nexus Monach и репутационную проверку движка. Она снижает риск, но не заменяет внимательную проверку адреса, особенно перед вводом пароля или платёжных данных. SHA-256 позволяет сверить загруженный файл с контрольной суммой издателя; наличие сертификата само по себе не означает, что файл безопасен.

## Автономный AI и Nexus Intelligence Fabric™

В браузере нет зависимости от Ollama и нет облачного резервного подключения. Full Offline release содержит собственные runtime и модели в папке `AI`: Qwen3 0.6B для быстрых структурированных задач анализа, Whisper base q5_1 для речи, multilingual-e5-small для смыслового индекса, SmolVLM 500M для изображений и три OPUS-MT ONNX-модели для перевода на русский. Qwen обслуживает один локальный `llama-server.exe`, доступный только через `127.0.0.1`. Перевод обслуживает отдельный постоянный Node/ONNX-процесс: он загружает OPUS-модели по требованию один раз и переиспользует их для страницы, выделения и видео. Runtime-код не содержит адресов облачных AI API.

Кнопка `RU` открывает два режима: перевод всей страницы и перевод звука видео в русские субтитры. Перевод выделенного текста находится только в контекстном меню страницы: выделите текст, нажмите правую кнопку и выберите «Перевести выделенное на русский». Перевод безопасно заменяет выбранный текстовый узел без отдельного окна; если выделение пересекает несколько DOM-узлов, Nexus оставляет страницу без изменений.

Звуковой режим использует один непрерывный WASAPI-сеанс и ограниченную очередь:
встроенный whisper-server загружает модель один раз, автоматически распознаёт
исходный язык без перевода на английский, после чего постоянный OPUS-конвейер
переводит реплику на русский. Qwen в переводе видео не участвует. Защищённое
DRM-видео или системные ограничения захвата звука могут не позволить получить
аудиопоток.

Стандартное окно Chromium DevTools создаёт Microsoft Edge Runtime. Nexus
открывает его без собственной AI-вкладки, не запускает для DevTools локальный
HTTP-мост и не собирает в фоне консольные события. Пункт контекстного меню
«Открыть в инструментах разработчика» по-прежнему передаёт выбранный DOM-узел
стандартной вкладке Elements.

Nexus Intelligence Fabric полностью открыт и находится в проекте `src/Nexus.Intelligence.Fabric`. Контракт `Nexus.Intelligence.Contracts` сохраняет модульную границу, но браузер и Fabric собираются одной командой и одной GitHub Actions цепочкой. Smart Omnibox/Nexus Search, глубокий анализ, углублённый многостраничный поиск, маршрутизация моделей и сводка агента входят в обычную сборку; Deep Page QA, Information Value Radar и дальнейшее развитие семантического графа будут добавляться в тот же открытый слой.

Обычный поиск не подменяется: поисковый провайдер видит запрос, потому что без
внешнего индекса найти страницы всего интернета невозможно, и пользователь
получает его привычную выдачу. После добровольного перехода на результат Nexus
ограниченно читает открытую страницу и до шести подходящих публичных страниц
того же сайта, локально готовит выжимку и показывает её справа. Следопыт не
заполняет формы, не переходит в аккаунт, корзину или оплату. При уходе с домена
панель закрывается, но несекретные метаданные исследования и выбранный источник
остаются в локальном графе знаний.

Guardian Center показывает отдельный локальный журнал Следопыта: технические
этапы, длительность и количество кандидатов/результатов. Сам запрос, URL, DOM,
текст страницы и значения форм туда не попадают; журнал не отправляется по сети.

SSH-панель не эмулирует Linux внутри браузера. Она запускает штатный
`%WINDIR%\System32\OpenSSH\ssh.exe`, принимает `user@host` и явный порт,
показывает время соединения и передаёт серверу только команды пользователя.
Для постоянной работы рекомендуется ключ и Windows `ssh-agent`.

Nexus Следопыт имеет один запуск: пользователь задаёт цель текстом и/или
фотографией и нажимает «Начать поиск». Глубокий анализ, ограниченный обход и
итоговая сводка являются внутренними стадиями Fabric, а не отдельными режимами.
Маршруты входа, профиля, корзины, оплаты и оформления заказа исключаются; агент
не заполняет формы и не совершает покупку.

Обычный и товарный поиск создают в Knowledge Constellation локальную цепочку
«запрос → источники/товары → выбранная страница». Сам текст страниц, DOM, cookies
и значения форм в граф не сохраняются; приватное окно не ведёт исследовательскую память.

Полная история улучшений и влияния изменений на приватность и сборку ведётся в `CHANGELOG.md`. Архитектура открытого исследовательского слоя описана в `docs/Nexus-Intelligence-Fabric.md`.

## Монитор приватности и прокси

Privacy Dock занимает всю нижнюю ширину обычного окна браузера и обновляется раз в пять минут. Панель можно временно скрыть кнопкой `×` либо включать и выключать через меню и настройки. Проверка работает локально: читает интерфейсы, шлюз и DNS Windows, ищет локальные признаки VPN/Tor/Proxy и собирает ICE-кандидаты WebRTC без STUN/TURN. Внешний IP, ASN и его достоверный регион намеренно не запрашиваются; региональная подпись — лишь оценка по настройкам Windows.

Full Offline release уже содержит whisper.cpp и `ggml-base-q5_1.bin`. Браузер не скачивает их при первом запуске и не создаёт сетевой зависимости для перевода звука.

Прокси задаётся как SOCKS5 либо HTTP с адресом и портом. Изменение требует перезапуска Nexus Monach. Список прямых исключений действует только на этот встроенный прокси. Он не способен отменить маршрут полноценного системного VPN: такое исключение необходимо настраивать в самом VPN-клиенте. Ограничение WebRTC от непроксируемого UDP включено по умолчанию; его можно отключить ради совместимости с отдельными видеозвонками и P2P-сервисами. Nexus Monach намеренно не подменяет часовой пояс, язык или User-Agent под географию IP.

## Сборка переносимой версии

Требования:

1. Windows 10/11 x64.
2. [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
3. Microsoft Edge WebView2 Runtime (в Windows 11 уже установлен).

Маленький source-архив не содержит сотни мегабайт весов моделей. Для пользовательской публикации собирается отдельный Full Offline архив с папкой `AI`; обычная исходная сборка явно предупредит, если этот комплект отсутствует.

Запустите `RUN_BUILD.cmd`. Скрипт создаст:

```text
dist/NexusMonach-Portable/
dist/NexusMonach-Portable-win-x64.zip
```

Архив самодостаточный по .NET: устанавливать .NET на компьютер пользователя не потребуется. WebView2 Runtime остаётся системным и автоматически получает исправления безопасности.

В portable-сборке `NexusMonach.exe` — проверяющий launcher Nexus Guardian, а
внутренний процесс браузера называется `NexusMonach.Browser.exe`. Инструкция по
созданию ключа целостности, Crash Vault и полной проверке находится в
`docs/Nexus-Guardian.md`.

В меню браузера доступен `Nexus Guardian · локальные рапорты`. Центр показывает
результат проверки целостности и сохранённые очищенные отчёты, позволяет создать
безопасный тестовый рапорт, скопировать или экспортировать его. Для нового
профиля отчёты по умолчанию хранятся только локально и не используют сеть.

Для разработки:

```powershell
dotnet restore NexusMonach.sln
dotnet run --project src/NexusMonach/NexusMonach.csproj
```

- portable-сборка: `NexusMonach-Portable/Data`;
- обычный запуск из Visual Studio: `%LOCALAPPDATA%\NexusMonach`;
- граф знаний: `knowledge-graph.json` в той же локальной папке `Data`; в приватном окне индексация не выполняется;
- приватное окно использует режим `IsInPrivateModeEnabled` и не записывает граф знаний и сессию;
- включение «Очищать данные сайтов при выходе» удаляет cookies, кэш и сохранённую сессию при закрытии обычного окна.

## Структура проекта

```text
src/Nexus.Intelligence.Contracts/  публичные интерфейсы и DTO локального Fabric
src/Nexus.Intelligence.Fabric/     открытая исследовательская оркестрация
src/Nexus.Guardian/                проверка целостности, crash-loop и безопасный запуск
src/Nexus.Guardian.Relay/          безопасная доставка отчётов тестировщиков в Matrix
src/NexusMonach/
  Assets/        эмблема и локальная стартовая страница
  Models/        настройки, вкладки и пользовательские данные
  Services/      профиль, защита, URL, локальный граф знаний, расширения
  Views/         главное окно и вспомогательные окна
scripts/         сборка и проверка
```

Весь собственный исходный код оболочки, `Nexus.Intelligence.Contracts` и `Nexus.Intelligence.Fabric` распространяется по MIT. Название и эмблема регулируются политикой `TRADEMARKS.md`, а сторонние runtime и модели сохраняют собственные лицензии.

Идея, видение и направление проекта — **CoderMonach**. Проектирование и код создаются совместно с **OpenAI Codex**.
